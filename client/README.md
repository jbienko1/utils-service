# utils-service — klient WWW

Osobna aplikacja (**Vite + TypeScript**) do korzystania z API z poziomu przeglądarki.

**Pełna dokumentacja:** [docs/README.md](docs/README.md) (instalacja i uruchomienie: [docs/install-and-run.md](docs/install-and-run.md)). **Dla agentów AI:** [docs/AGENTS.md](docs/AGENTS.md).

Skrót:

```powershell
cd client
npm install
npm run dev
```

Uruchom wcześniej backend FastAPI (np. port 8000) — zobacz [README.md](../README.md) repozytorium.
